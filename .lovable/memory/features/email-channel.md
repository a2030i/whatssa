---
name: Email Channel
description: Full email integration (SMTP send + IMAP receive) within unified inbox
type: feature
---

دمج البريد الإلكتروني كقناة تواصل كاملة (SMTP/IMAP) داخل صندوق الوارد الموحد: 
1. الإعدادات: دعم حسابات متعددة مع إعدادات جاهزة للمزودين (Gmail, Outlook, Zoho) وأدلة تفاعلية لاستخراج كلمات مرور التطبيقات (App Passwords).
2. المزامنة: يختار المستخدم بين مزامنة 'الرسائل الجديدة فقط' أو 'القديمة والجديدة' (آخر 30 رسالة).
3. الاستقبال: وظيفة Edge (email-fetch-imap) تجلب الرسائل الواردة عبر IMAP وتحفظها كمحادثات. تعمل تلقائياً كل دقيقتين عبر pg_cron + زر يدوي "جلب الوارد" في الإعدادات.
4. الإرسال: وظيفة Edge (email-send) ترسل عبر SMTP وتحفظ المحادثة.
5. المراسلة والظهور: ميزة 'إنشاء إيميل' (Compose) من داخل الصندوق، مع تبويب فلتر مخصص "إيميل" (Emails) في قائمة المحادثات لتمييزها عن الواتساب بجانب أيقونة البريد (📧).
6. التوجيه: إمكانية إسناد موظف أو فريق مخصص لكل حساب بريد آلياً.
7. ملاحظة: يتطلب Gmail استخدام 'App Password'. القاعدة الخارجية لا تحتوي على عمود last_message_sender في conversations.
