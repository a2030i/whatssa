# Memory: features/ai/knowledge-base-auto-reply
Updated: now

نظام الرد التلقائي الذكي المدعوم بقاعدة معرفة (AI Auto-Reply):
1. **قاعدة معرفة لكل قناة**: يمكن ربط محتوى المعرفة بقنوات محددة (مثلاً: معلومات المبيعات لرقم المبيعات فقط) مع fallback للمحتوى العام.
2. **تفعيل AI لكل قناة**: زر تفعيل/تعطيل في إعدادات كل قناة مع تحكم بحد المحاولات وكلمات التحويل الفوري.
3. **حماية من الهلوسة**: JSON structured output مع `found_in_knowledge` + `confidence` + temperature 0.1 لضمان عدم اختراع إجابات.
4. **حد المحاولات**: بعد X رد من AI بدون إجابة واضحة → تحويل تلقائي لموظف (قابل للتخصيص لكل قناة).
5. **كلمات التحويل الفوري**: العميل يكتب "موظف" أو "بشري" → تجاوز AI فوراً.
6. **الأسئلة المعلقة**: عند عدم وجود إجابة، AI يولّد أسئلة للمدير → المدير يجيب → تُحفظ تلقائياً في قاعدة المعرفة.
7. **سجل الشفافية**: توثيق خطوات AI (تجاوز، تحويل، حد أقصى) كرسائل نظام داخل المحادثة.
8. **حلقة التعلم**: سؤال عميل → AI لا يعرف → أسئلة للمدير → إجابة → قاعدة المعرفة تتحدث → AI يرد مستقبلاً.
