# Respondly — Project Instructions

## Project Info
- **Name**: Respondly
- **Stack**: React + TypeScript + Vite + Supabase

## Versioning
- Every commit message must include a version number in the format `v1.x.x`
- Start from `v1.0.1` and increment the patch number for each commit
- Example: `fix: support env var fallback v1.0.1`
